# 硬件参考图

> 板子到货后拍摄的参考照片，用于确认芯片型号、GPIO 丝印、接口布局。

## 已归档

| 文件 | 内容 | 大小 | 用途 |
|------|------|------|------|
| `board_front.jpg` | 主板正面（带丝印） | 1.9MB | 确认 GPIO 标号、芯片型号、DVP 并口位置 |
| `board_back.jpg` | 主板背面（带丝印） | 1.7MB | 确认背面走线、TF 卡槽、模组丝印 |
| `accessories.jpg` | 套件配件全貌 | 1.7MB | LCD、摄像头、扬声器、麦克风、排线 |

- `board_*.jpg` — 主板照片
- `chip_*.jpg` — 芯片特写
- `pinout_*.jpg` — 引脚相关参考
- `accessories.jpg` — 配件
- `manual_*.pdf` — 说明书 PDF
