### T8: 代码文件最终静态自检

不修改代码，只做检查：
1. 确认 main/ 下所有 .h 有 include guard (`#ifndef` / `#define`)
2. 确认 main/ 下所有 .cpp 包含了对应的 .h
3. 统计: 总文件数 / 总行数 / 模块数

```bash
# 检查 include guards
grep -l "#ifndef ROBOMIND_" main/*.h

# 检查 .cpp → .h 配对
for f in main/*.cpp; do
    h="main/$(basename "$f" .cpp).h"
    if [ -f "$h" ]; then
        grep -q "#include \"$(basename "$h")\"" "$f" && echo "✅ $f → $h" || echo "❌ $f missing include for $h"
    fi
done
```
