## F9 🟠 Connect() 中间步骤失败后资源未完全清理

**场景**: `esp_wifi_init` 成功后，若后续步骤 (set_mode/set_config/start) 失败 → 已创建的 event group、已注册的事件处理器、已 init 的 wifi 不会清理。

**修复**: 添加错误清理路径:

```cpp
bool WifiManager::Connect() {
    // ... 前置初始化 ...
    
    // WiFi init
    esp_err_t ret = esp_wifi_init(&cfg);
    if (ret != ESP_OK) {
        ESP_LOGE(TAG, "wifi_init failed: %s", esp_err_to_name(ret));
        return false;  // 此时尚未分配资源
    }
    
    // === 从此处开始需要错误清理 ===
    
    // 注册事件处理器
    ret = esp_event_handler_instance_register(
        WIFI_EVENT, ESP_EVENT_ANY_ID,
        &WifiManager::EventHandler, this, &s_wifi_event_handler);
    if (ret != ESP_OK) {
        ESP_LOGE(TAG, "wifi event handler register failed");
        esp_wifi_deinit();  // ← 清理 wifi
        return false;
    }
    
    // ... set_mode / set_config / start ...
    
    // 如果 wifi_start 失败，需要清理 event handler + wifi
    ret = esp_wifi_start();
    if (ret != ESP_OK) {
        ESP_LOGE(TAG, "wifi_start failed");
        esp_event_handler_instance_unregister(
            WIFI_EVENT, ESP_EVENT_ANY_ID, s_wifi_event_handler);
        esp_event_handler_instance_unregister(
            IP_EVENT, IP_EVENT_STA_GOT_IP, s_ip_event_handler);
        esp_wifi_deinit();
        return false;
    }
    
    // ... 正常流程 ...
}
```

核心原则: **每个分配点之后的所有错误路径都要逆序清理已分配资源。**

## F16 🟡 Disconnect() 缺 esp_wifi_deinit 配对

```cpp
void WifiManager::Disconnect() {
    // 注销事件处理器
    if (s_wifi_event_handler) {
        esp_event_handler_instance_unregister(
            WIFI_EVENT, ESP_EVENT_ANY_ID, s_wifi_event_handler);
        s_wifi_event_handler = nullptr;
    }
    if (s_ip_event_handler) {
        esp_event_handler_instance_unregister(
            IP_EVENT, IP_EVENT_STA_GOT_IP, s_ip_event_handler);
        s_ip_event_handler = nullptr;
    }
    
    esp_wifi_disconnect();
    esp_wifi_stop();
    esp_wifi_deinit();  // ← 新增: 释放 WiFi 驱动资源
    
    if (s_wifi_event_group) {
        vEventGroupDelete(s_wifi_event_group);
        s_wifi_event_group = nullptr;
    }
    
    state_ = WifiState::kDisconnected;
}
```
